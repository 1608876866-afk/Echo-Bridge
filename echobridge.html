知情同意书 & 隐私保护须知

1. 数据存储：你的所有数据存储在浏览器本地（LocalStorage），不会上传至任何服务器。

2. 数据风险：清除浏览器缓存或使用无痕模式将导致数据丢失，建议定期导出备份。

3. AI 服务：部分功能需调用 DeepSeek API，你需要自行提供 API Key。对话内容通过 API 传输，请勿输入过于敏感的个人身份信息。

4. 适用范围：本产品适用于轻中度社交焦虑的自我辅助训练，不适用于精神分裂症、重度抑郁发作期或有自伤/自杀风险的人群。

5. 非医疗建议：本产品不能替代专业医疗诊断和治疗。

6. 你的权利：你可以随时导出、删除自己的数据，拥有对自己数据的完全控制权。
