欢迎使用共鸣之桥。在开始之前，请仔细阅读以下信息：

1. 数据存储：你的所有数据（包括测评结果、日记、聊天记录等）均存储在浏览器本地（LocalStorage），不会上传至任何服务器。

2. 数据风险：清除浏览器缓存、使用无痕模式或更换浏览器将导致数据丢失。建议定期导出数据备份（可在设置中进行）。

3. AI 服务：部分功能（如对话引导、日记分析）需要调用 DeepSeek API，你需要自行提供 API Key。你的对话内容会通过 API 传输，请勿输入过于敏感的个人身份信息。

4. 适用范围：本产品适用于轻中度社交焦虑的自我辅助训练。不适用于精神分裂症、重度抑郁发作期、或有自伤/自杀风险的人群。如果你正处于危机中，请立即联系专业机构。

5. 非医疗建议：本产品提供的所有内容仅供参考和辅助训练，不能替代专业医疗诊断和治疗。

6. 你的权利：你可以随时导出、删除自己的数据，也可以随时停止使用。你拥有对自己数据的完全控制权。
